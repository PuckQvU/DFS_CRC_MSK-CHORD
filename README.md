# Project Description

In this project, I will be focusing on predicting progression-free survival (PFS) and overall survival (OS) after the first surgery in patients with stage I-III colorectal cancer. I will assess different models and combinations of features to find the optimal prediction model. 

# Data

This project is uses the Memorial Sloan Kettering clinicogenomic, harmonized oncologic real-world dataset (MSK-CHORD), published by [Jee *et al*. (2024)](https://doi.org/10.1038/s41586-024-08167-5 "Automated real-world data integration improves cancer outcome prediction"). This dataset is a collection of clinicogenomic data and structured real-world data (RWD), captured from unstructured medical text using natural language processing (NLP). The dataset includes data from non-small-cell lung (n = 7,809), breast (n = 5,368), colorectal (n = 5,543), prostate (n = 3,211) and pancreatic (n = 3,109) cancers. The full MSK-CHORD dataset was downloaded from [cBioPortal](https://www.cbioportal.org/study/summary?id=msk_chord_2024 "cBioPortal: MSK-CHORD") on the 10th of June, 2025.
